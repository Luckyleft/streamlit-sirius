This is a fork of parker-stephens's original siriusxm-activator project. It can activate a Sirius radio for three months at a time. It does so by replicating the API calls that the SXM dealer app makes when activating a radio.

It will ask for your radio ID, then try and activate it.

The purpose of this fork is to make it compatible with the free version of Streamlit. You can set up your own or use this one

The bulk of the code has remained unchanged. This way, if the original changes, updating this fork should be easy. The major changes are at the bottom, where I've added a lot of comments and the code for the Steamlit app.
